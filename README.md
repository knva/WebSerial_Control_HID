## Serial or WebSerial Control HID DEMO


![altext](https://github.com/knva/WebSerial_Control_HID/blob/master/images/show.gif)

使用串口或者WEBUSB操作HID设备

```
shoot\r -> mouse.click();
move100:100\r - > mouse.move(100,100);
```
板卡使用树莓派PICO

库使用 https://github.com/earlephilhower/arduino-pico/releases/download/global/package_rp2040_index.json

usb栈为tinyUsb

开发工具为Arduion IDE 2.0.2
